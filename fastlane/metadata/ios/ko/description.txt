Libellus Potionis는 프라이버시를 중시하는 무료, 오픈 소스, 광고 없는 음주 기록 도구입니다. 완전히 오프라인으로 음주를 모니터링하고 조절하며 관리하도록 도와줍니다. 카메라, 마이크, 위치 등 침습적인 권한이 전혀 필요 없으며, 네트워크 접근 없이 작동합니다.

주요 기능

• 똑똑한 기록: 나만의 음료를 정의하거나 국제적으로 흔한 프리셋에서 선택할 수 있습니다. 음료를 즉시 또는 정확한 시간 보정과 함께 사후에 기록할 수 있습니다.
• 여러 한도를 동시에: 세 가지 한도를 병행해 설정합니다 — 일일 한도(순수 알코올 그램), 7일 이동 주간 한도(그램), 주당 최대 음주 일수. 진행 막대가 실시간으로 알려 줍니다.
• 혈중알코올농도(BAC) 추정: 체중을 입력하면 정립된 Widmark 공식을 바탕으로 한 실시간 근삿값을 얻을 수 있습니다.
• 상담용 보고서: 상담 및 카운슬링 예약을 위해 설계된, 명확하고 잘 정리된 두 페이지 분량의 PDF 보고서를 생성하여 습관을 통계적으로 요약합니다.
• 데이터 이동성: 전체 데이터셋을 표준 CSV 파일로 내보내 외부 처리(예: LibreOffice Calc)에 사용하거나, 기기 간에 데이터를 옮기기 위한 안전한 JSON 백업을 만들 수 있습니다.
• 세밀한 조정: "하루 시작" 시각을 맞춤 설정하여 늦은 밤의 음료가 올바른 저녁에 집계되도록 하고, 깔끔한 재시작을 위해 사용자 지정 평가 시작일을 설정할 수 있습니다.

앱 안에 자세한 사용 설명서가 제공됩니다.

개인정보 보호 및 보안

Libellus Potionis는 절대적인 데이터 주권과 엄격한 데이터 최소화를 중심으로 설계되었습니다. 네트워크 접근을 완전히 배제한 최소 권한 프로필로 작동하므로, 개인 데이터가 기기를 벗어나는 일이 없습니다. 데이터는 앱의 비공개 저장소에 보관되며, 저장 시 iOS Data Protection(하드웨어 기반 파일 암호화)과 앱 샌드박스로 보호됩니다. 선택적 Face ID 또는 Touch ID 잠금은 무단 물리적 접근으로부터 보호합니다. 이 오프라인 우선 방식은 추적, 클라우드 동기화, 외부로의 데이터 유출을 제거합니다.

호환성

이 앱은 iOS 17 이상의 iPhone에서 작동합니다. 앱 내의 완전히 독립적인 언어 선택기를 통해 시스템 언어와 무관하게 언제든지 앱 언어를 전환할 수 있습니다.
