It is used for third-party assets bundled in this repository whose authors have
released them into the public domain, and for which therefore no license text
exists. REUSE (https://reuse.software) still requires every file to carry a
license identifier and every referenced identifier to have a text in LICENSES/;
this file satisfies that requirement for such public-domain assets.
